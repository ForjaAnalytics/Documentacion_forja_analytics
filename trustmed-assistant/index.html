Trustmed Assistant

Plataforma SaaS para cuidados paliativos que graba la consulta médica, la transcribe con ElevenLabs Scribe, extrae datos clínicos estructurados con Claude y devuelve una historia editable lista para exportar. Multi-tenant, con roles, cuotas y panel de administración de plataforma.

Python 3.11+ · FastAPI React 18 · Vite · Tailwind Postgres + SQLAlchemy Google Drive (resumable) ElevenLabs Scribe Claude tool use
