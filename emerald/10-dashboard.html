Dashboard

SPA en web/ desplegada en Vercel. React 18 + TypeScript + Vite + Tailwind, con Recharts para gráficos, framer-motion para animación y lucide para iconos.

Páginas (~16)

Overview · Analytics · Groups + GroupDetail · Pipeline · AIInterview · CandidateDetail · ActionQueue · Decisions · Activity · Search · CVManager · Workers · Settings · Users · Login.

Arquitectura front

Routing por hash (useHashRoute), polling de queries (usePolledQuery), eventos en vivo (useLiveEvents), cliente API centralizado (lib/api.ts) y auth por cookie (lib/auth.ts). ~50 componentes reutilizables (KPIs, charts, command palette, timelines).

CORS cross-site. El dashboard vive en Vercel y la API en Railway (dominios distintos). Las cookies son HttpOnly · Secure · SameSite=None y CORS usa allow_credentials=true con orígenes explícitos (nunca * con credenciales — el browser lo rechaza). Se configura vía EMERALD_CORS_ORIGINS.
