09 · Backend

API REST

FastAPI. Endpoints públicos para health/observabilidad y 15 routers de dashboard bajo /api/*, todos protegidos por un JWT en cookie httpOnly aplicado globalmente como dependencia.

Endpoints públicos / operacionales

MétodoRutaDescripción
GET/healthLiveness — {"status":"ok"}.
GET/statusConteos por sync_status de grupos/jobs/applicants + último poll.
GET/pendingPrimeros N grupos y applicants pendientes.
GET/eventsÚltimos N eventos del audit log.
POST/pollDispara un ciclo de polling on-demand (lock evita concurrencia).
POST/advanceCiclo manual del advancer (legacy).
GET/groups-waiting-assessmentRoles creados pero esperando configuración humana de plantilla.
